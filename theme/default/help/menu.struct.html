结构

├── app
应用入口
├──── common
  公共文件（函数、工具集等）
├──── conf
  配置文件
├──── controller
  控制器
├──── data
  存放插件数据
├──── directive
  自定义标签
├──── filter
  过滤器
├──── plugin
  外部插件
├──── service
  业务服务
├──── app.js
  全局配置/路由配置
├──── main.js
  RequireJS配置
├── lib 库文件
├──── angular
  MVC框架
├──── angular-ui
  Angular扩展
├──── bootstrap
  UI框架
├──── requirejs
  模块化加载
├──── underscore
  JS工具类库
├── static 资源文件目录
├──── common
  公共资源
├────── css
    样式文件
├────── images
    图片文件
├────── js
    脚本文件
├──── ...
  按照主题存放资源文件
├── theme 主题模板目录
├──── ...   按照主题存放模板文件
└── index.html 默认单页入口
